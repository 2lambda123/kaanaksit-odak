# odak.wave.wavenumber

`wavenumber(wavelength)`

Definition for calculating the wavenumber of a plane wave
 
**Parameters:**

    wavelength   : float
                   Wavelength of a wave in mm.
                       
**Returns**

    k            : float
                   Wave number for a given wavelength.
