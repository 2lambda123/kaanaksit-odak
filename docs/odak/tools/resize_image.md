# odak.resize_image

`resize_image(img,target_size)`

Definition to resize a given image to a target shape.
 
**Parameters:**
                       
    img           : ndarray
                    MxN image to be resized.
    target_size   : list
                    Target shape.

**Returns**

    num        : float
                 Size in new unit.
    x          : str
                 New unit bytes, KB, MB, GB or TB.
