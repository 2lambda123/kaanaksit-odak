# odak.convert_bytes

`convert_bytes(num)`

A definition to convert bytes to semantic scheme (MB,GB or alike). 
Inspired from `https://stackoverflow.com/questions/2104080/how-can-i-check-file-size-in-python#2104083`.
 
**Parameters:**
                       
    num        : float
                 Size in bytes

**Returns**

    num        : float
                 Size in new unit.
    x          : str
                 New unit bytes, KB, MB, GB or TB.
