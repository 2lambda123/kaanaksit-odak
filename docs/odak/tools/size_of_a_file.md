# odak.size_of_a_file

`size_of_a_file(file_path)`

A definition to get size of a file with a relevant unit.
 
**Parameters:**

    file_path  : float
                 Path of the file.
**Returns**

    a          : float
                 Size of the file.
    b          : str
                 Unit of the size (bytes, KB, MB, GB or TB).
