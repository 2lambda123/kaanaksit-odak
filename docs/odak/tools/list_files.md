# odak.list_files

`list_files(path,key='*.*',recursive=True)`

Definition to list files in a given path with a given key.
 
**Parameters:**
                       
    path        : str
                  Path to a folder.
    key         : str
                  Key used for scanning a path.
    recursive   : bool
                  If set True, scan the path recursively.

**Returns**

    files_list  : ndarray
                  list of files found in a given path.
